THE MAZE PROJECT

Creating a Game in 3D using raycasting. Designed for kids so they can improve their spatial recognition and analytical skills.

Download and try the game on Linux

ALL ABOUT THE MAZE PROJECT

At the end of our foundation classes at ALXSE we had to create my own portfolio project. The Maze-Game project was chosen to learn about game development. We were interested in developing with the programming languages... we had learned and wanted to learn new methods and techniques that could be implemented in software development. Then we decided to learn about Raycasting tool and implement it using SDL2 library to make this game. It's interesting to know how a game, like Wolfenstein 3D, was able to implement pseudo-3D that revolutionized the gaming industry and opened the way for other game engines and technologies.
